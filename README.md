## SecureVault_java
A password-protected desktop file vault built in Java Swing with AES-256 encryption. Store, hide and manage your files securely.

## ✨ Features
- 🔐 AES-256 Encryption
- 📁 All file types supported
- 🛡 Password protected vault
- 🗑 Auto-delete after export
- 💻 Clean modern UI

## 🚀 How to Run
1. Install Java JDK & NetBeans
2. Clone this repo
3. Open project in NetBeans
4. Then run it

## 🛠 Tech Stack
- Java Swing
- AES-256 / CBC Encryption
- - SHA-256 Password Hashing
- Apache NetBeans IDE
